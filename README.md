# Bug 冲刺 H5 标准样板

这是后续 Cocos Creator 3.8 / LayaAir 3.x / Godot 4.6 复刻对比的纯 H5 标准版本。

## 玩法

控制程序员左右移动，躲避 Bug、需求变更、加急、线上报错，收集咖啡、键盘、代码块、提审通过道具。

## 统一规则

- 画布尺寸：750 x 1334 竖屏
- 玩家初始血量：3
- 障碍物：Bug / 需求变更 / 加急 / 线上报错
- 道具：咖啡 / 键盘 / 代码块 / 提审通过
- 分数：存活时间 + 道具加分
- 结束：血量归零

## 操作

- 手机：手指左右滑动
- PC：鼠标拖动 / 方向键
- 开始或重来：点击按钮区域 / 空格键

## 后续对比指标

- 建项目耗时
- 核心玩法完成耗时
- UI 制作体验
- 动画制作体验
- 调试体验
- Web 导出耗时
- 首包大小
- 首屏加载时间
- 手机浏览器帧率
- 最大坑点
